TOKYO MEAT MAP

都内の“肉に本気”なお店をマッピング

ホットペッパーから取得した焼肉・ステーキ等の肉系店舗を集約し、地図上で一覧表示します。毎回の取得結果はCSVに出力され、GitHub Pagesで確認できます。

マップを開く

使い方

  • 上の「マップを開く」をクリック
  • ランク・ジャンル・検索で絞り込み
  • リストから店舗を選んでズーム
  • 現在地ボタンで自分の近くを確認

データ更新

  • スクリプト: python -m meatmap.cli --copy-to-docs
  • データソース: ホットペッパー（東京、肉系ジャンル＋キーワード）
  • CSV先頭に生成日時・件数をメタ行で付与

注意・免責

  • 情報の正確性は保証しません
  • 広告・寄付等は未設定（必要に応じて追加予定）
  • APIキーは公開しないでください

サポート / 広告プレースホルダ

  • AdSense: 審査後にコード挿入予定（マップ下部に配置）
  • 寄付: BuyMeACoffee / Ko-fi などのボタンを配置予定
  • アフィリエイト: 関連リンクがあればマップ説明に掲示
