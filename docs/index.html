TOKYO MEAT MAP

都内の“肉に本気”なお店をマッピング

ホットペッパーグルメ Webサービス由来の店舗データと、従前から保有する未検証の旧ローカルデータから、焼肉・ステーキ等の肉系店舗を地図上で探せます。

マップを開く

使い方

  • 上の「マップを開く」をクリック
  • ジャンルを選ぶか、店名で検索して絞り込み
  • ピンを選んで店舗情報と経路を確認
  • 現在地ボタンで自分の近くを確認

データ更新

  • 現行APIデータ更新: make csv（公開前に旧データとの統合・検証を実施）
  • 現行データ: ホットペッパーグルメ Webサービス
  • 旧データ区分: 旧ローカルデータ（未検証）
  • 現行APIで確認できない旧Hot Pepper行も旧データ（未検証）として保持
  • CSVに生成日時、各行に取得・最終確認日または記録日と確認状態を記録

注意・免責

  • 情報の正確性は保証しません
  • 「旧データ（未検証）」は現在の営業状況等を再確認していません
  • 本サイトは各データ提供元・参照先の公式または提携サービスではありません
  • 来店前に店舗の公式情報または店舗へ直接ご確認ください
  • 現在、広告配信・寄付受付・アクセス解析は行っていません
  • APIキーは公開しないでください

修正依頼・ご意見

  • 閉店・住所違い・表示不具合をご報告ください
  • GitHub Issues は公開投稿です
