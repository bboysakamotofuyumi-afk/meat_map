東京都 / 肉料理店マップ（S/A 公開中・B/C は今後拡張予定）

このマップについて

ホットペッパーなどから取得した「肉料理が強い店」をカーニボアスコアでランク付けし、マップ表示しています。

  • 対象エリア: 東京都（23区＋多摩）
  • 対象カテゴリ: 焼肉 / ホルモン / ステーキ / シュラスコ / ジビエ など肉メインの飲食店
  • 現在は S/A ランク中心を公開（B/C は更新環境が整い次第拡張）
  • 地図機能: 検索・ジャンル絞り込み・並び替え・住所検索・現在地表示をサポート

データ更新

マップで利用する店舗データは、あらかじめ生成したCSVを読み込んで表示しています。

最終更新: -
件数: -

ネットワークが利用できる環境で python -m meatmap.cli --output output/meatmap.csv --copy-to-docs を実行すると最新データを再生成できます。

利用上の注意

  • 営業時間・住所・閉店情報は変わる場合があります。来店前に公式情報をご確認ください。
  • 位置情報はブラウザ内でのみ使用し、サーバーには送信しません。
  • マップ表示やCSV取得に失敗した場合はパス（クエリ）を確認し、時間をおいて再度アクセスしてください。
