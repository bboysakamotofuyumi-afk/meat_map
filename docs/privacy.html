プライバシーポリシー

TOKYO MEAT MAP（以下「当サイト」）は、地図上で飲食店を閲覧するためのデモサイトです。位置情報や広告に関する方針を以下のとおり定めます。

1. 取得する情報

  • 現在地の取得（任意）: 「現在地」ボタンを押した場合のみブラウザの Geolocation API で取得します。取得した位置情報はブラウザ内で地図表示にのみ使用し、サーバーへ送信・保存しません。
  • アクセスログ: GitHub Pages 等のホスティングが提供する一般的なアクセスログが生成される場合があります。

2. 位置情報の扱い

位置情報はブラウザ内で地図表示にのみ利用します。サーバーへ送信したり、第三者へ共有したりすることはありません。

3. 広告・Cookie

現在、広告コードは未設置です。将来 Google AdSense などの広告サービスを利用する場合、Cookie や類似技術が使用され、利用者のブラウザに保存されることがあります。導入時に本ポリシーを更新し、必要に応じて同意取得を行います。

4. 外部サービス

  • 地図表示: Leaflet / OpenStreetMap タイルを使用しています。
  • ジオコーディング: Nominatim(OpenStreetMap) を使用しています。
  • データ取得元: ホットペッパーグルメ API から取得したデータを加工・掲載しています。

5. お問い合わせ

問い合わせ先は未設定です。運営者のメールアドレスまたはSNSアカウントを決定後、ここに記載してください。

本ポリシーは必要に応じて改定します。重要な変更がある場合はサイト上で告知します。
