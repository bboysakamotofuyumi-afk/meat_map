プライバシーポリシー

TOKYO MEAT MAP（以下「当サイト」）は、地図上で飲食店を閲覧するための個人開発サイトです。位置情報や外部サービスに関する方針を以下のとおり定めます。

1. 取得する情報

  • 現在地の取得（任意）: 「現在地」ボタンを押し、ブラウザで許可した場合のみ Geolocation API で取得します。当サイトのサーバーへの送信・保存や、ブラウザのローカルストレージへの保存は行いません。
  • アクセスログ: GitHub Pages 等のホスティングが提供する一般的なアクセスログが生成される場合があります。

2. 位置情報の扱い

取得した緯度・経度そのものを当サイトのサーバーへ送信・保存することはありません。ただし、地図表示や外部リンクの利用に伴い、各提供者へIPアドレス、ブラウザ情報、表示範囲またはリンク先に必要な情報等が送信される場合があります。

3. 広告・Cookie

当サイト運営者は現在、広告配信およびアクセス解析を行っておらず、これらの目的で Cookie 等を使用していません。将来導入する場合は、本ポリシーを事前に更新し、必要な案内・同意取得を行います。表示高速化のため、Service Worker のキャッシュと店舗住所のジオコーディング結果をブラウザ内に保存する場合があります。

4. 外部サービスと掲載データ

  • ホスティング: GitHub Pages
  • 地図表示: Leaflet / OpenStreetMap タイル
  • 住所の座標補完: Nominatim（OpenStreetMap）
  • 画面ライブラリ配信: jsDelivr / cdnjs
  • 現行の店舗データ: ホットペッパーグルメ Webサービス
  • 旧ローカルデータ（未検証）: 過去にローカルで保有していた店舗情報（この区分だけの行には外部店舗ページへのリンクを表示しません）
  • 利用者が明示的に開く外部リンク: 店舗ページ、Google マップ、X、Facebook、GitHub

5. お問い合わせ

掲載情報の修正依頼・不具合報告・ご意見は GitHub Issues からお知らせください。公開投稿となるため、個人情報は記載しないでください。
